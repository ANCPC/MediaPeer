# MediaPeer

> A lightweight, open-source, peer-to-peer media sharing platform focused on creators, privacy, and community.

**Project Status:** Early Development (v0.0.1)

---

# Vision

MediaPeer aims to provide a modern media platform without relying on intrusive advertising or excessive data collection.

The project combines a lightweight server with peer-to-peer distribution technologies to reduce infrastructure costs while giving creators more control over their content.

Core principles:

- Open Source
- Privacy First
- Community Driven
- Lightweight
- Creator Friendly
- Cross Platform
- Self Hostable (future)

# Philosophy

MediaPeer is designed around a simple idea:

> Users should own their experience.

The platform prioritizes privacy, transparency, performance, and long-term sustainability over engagement algorithms.

Creators should be supported directly by their communities instead of relying on intrusive advertising.
